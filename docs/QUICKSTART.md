Get the app running in under 2 minutes.

## Prerequisites

- **Node.js** 18+ ([nodejs.org](https://nodejs.org))
- **npm** (included with Node)

## Steps

### 1. Install dependencies

```bash
cd solo-entrepreneur-curriculum
npm install
```

### 2. Run the dev server

```bash
npm run dev
```

### 3. Open in browser

Go to: **http://localhost:3000**

---

## First actions

| Action | Where |
|--------|--------|
| See all categories | Home page (`/`) |
| Start learning | Click **Category 1: Founder Readiness & Mindset** |
| View full curriculum at once | Click **Full curriculum** in the header or go to `/all` |
| Jump to a category | Click any category card on the home page |

---

## Build for production (optional)

```bash
npm run build
npm start
```

Runs the app on port 3000 (or `PORT` env). Use a process manager (e.g. PM2) or host on Vercel/Netlify for production.
